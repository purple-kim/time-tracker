# Time Tracker

고양이 캐릭터가 데스크탑 위에 떠서 작업 시간을 보여주는 작은 타임 트래커 앱입니다.

작업을 시작하면 고양이가 노트북을 타이핑하고, 작업 시간이 흐르는 동안 자연스럽게 표정과 액션이 바뀝니다. 마우스로 원하는 위치에 옮겨두고 데스크탑 위젯처럼 사용할 수 있습니다.

## 주요 기능

- 작업 시간 측정
- 데스크탑 위에 떠 있는 투명 위젯
- 마우스 드래그로 위치 이동
- 고양이 타이핑 애니메이션
- 작업 흐름에 따라 바뀌는 표정과 액션
- 다음 캘린더 일정 표시

## 앱 설치

일반 사용자는 GitHub Releases에서 macOS용 앱 파일을 내려받아 사용합니다.

1. Releases 페이지에서 최신 버전의 `Time Tracker` 앱 파일을 다운로드합니다.
2. 압축 파일이면 압축을 풉니다.
3. `Time Tracker.app`을 `Applications` 폴더로 옮깁니다.
4. 앱을 실행합니다.

macOS에서 보안 경고가 뜨면 `시스템 설정 > 개인정보 보호 및 보안`에서 실행을 허용해야 할 수 있습니다.

## 개발자 실행

프로젝트를 직접 실행하거나 앱 파일을 만들 때만 아래 명령어를 사용합니다.

의존성을 설치합니다.

```bash
npm install
```

로컬에서 앱을 실행합니다.

```bash
npm start
```

macOS 앱 번들을 생성합니다.

```bash
npm run pack:mac
```

생성된 앱은 `dist/mac-arm64/Time Tracker.app`에서 확인할 수 있습니다.

## 캘린더 데이터

캘린더 일정은 `calendar-data.js`에서 읽습니다.

다른 사용자가 이 앱을 사용할 때는 자신의 캘린더 데이터를 연결해야 합니다. 저장소에는 개인 캘린더 일정이 포함되어 있지 않습니다.

현재 `calendar-data.js`는 예시 구조만 제공합니다.
