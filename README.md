# Time Tracker

Time Tracker는 데스크탑 위에 작은 픽셀 고양이를 띄워두고 작업 시간을 확인할 수 있는 macOS 앱입니다.

고양이는 노트북을 타이핑하는 모습으로 움직이고, Google Calendar를 연결하면 다음 일정과 일정 시작 전 알림도 받을 수 있습니다.

## 주요 기능

- 작업 시간 타이머
- 데스크탑 위에 떠 있는 픽셀 고양이
- 고양이 테마 선택: 치즈, 까망, 삼색이, 샴
- 타이핑 소리 켜기/끄기
- Google Calendar 연결
- 다음 일정 표시
- 일정 시작 10분 전, 5분 전 알림
- 알림 시 고양이 울음소리 재생

## 앱 다운로드하기

Time Tracker는 macOS 앱 파일로 사용할 수 있습니다.

1. [Time Tracker 다운로드 페이지](https://github.com/purple-kim/time-tracker/releases)에 들어갑니다.
2. 가장 위에 있는 최신 버전을 엽니다.
3. `Time-Tracker-...dmg` 파일을 다운로드합니다.

   본인 Mac에 맞는 파일을 선택해 주세요.

   - Apple Silicon Mac: `Time-Tracker-0.1.1-arm64.dmg`
   - Intel Mac: `Time-Tracker-0.1.1-x64.dmg`

4. 다운로드한 `.dmg` 파일을 더블 클릭합니다.
5. 열린 창에서 `Time Tracker` 앱을 `Applications` 폴더로 드래그합니다.
6. `Applications` 폴더에서 `Time Tracker`를 실행합니다.

처음 실행할 때 macOS에서 보안 안내가 나올 수 있습니다.

그럴 때는 아래 순서로 허용해 주세요.

1. Mac의 `시스템 설정`을 엽니다.
2. `개인정보 보호 및 보안`으로 이동합니다.
3. Time Tracker 실행을 허용합니다.
4. 다시 Time Tracker를 실행합니다.

## 사용 방법

고양이 위젯 위에서 마우스 오른쪽 버튼을 누르면 메뉴가 열립니다.

메뉴에서 아래 기능을 사용할 수 있습니다.

- `고양이 테마`: 원하는 고양이 테마를 선택합니다.
- `타이핑 소리`: 타이핑 소리를 켜거나 끕니다.
- `캘린더 연결`: Google Calendar를 연결합니다.
- `캘린더 연결 해제`: 연결된 Google Calendar를 해제합니다.
- `시작`: 작업 시간을 재기 시작합니다.
- `일시정지`: 작업 시간을 잠시 멈춥니다.
- `끝내기`: 앱을 종료합니다.

고양이를 드래그하면 원하는 위치로 옮길 수 있습니다.

## Google Calendar 연결하기

Google Calendar 연결은 선택 기능입니다.
연결하지 않아도 타이머와 고양이 위젯은 사용할 수 있습니다.

연결 방법은 아래와 같습니다.

1. Time Tracker를 실행합니다.
2. 고양이 위젯 위에서 마우스 오른쪽 버튼을 누릅니다.
3. `캘린더 연결`을 선택합니다.
4. Google 로그인 화면이 열리면 본인 계정을 선택합니다.
5. 권한 안내 화면이 나오면 내용을 확인한 뒤 계속 진행합니다.
6. Calendar 읽기 권한을 허용합니다.
7. 연결이 완료되면 다음 일정이 말풍선에 표시됩니다.

Time Tracker는 일정을 수정하지 않습니다.
다음 일정을 보여주고 알림을 보내기 위해 Calendar 읽기 권한만 사용합니다.

한 번 연결하면 앱을 껐다가 다시 켜도 연결 상태가 유지됩니다.

## 일정 알림

Google Calendar를 연결하면 일정 시작 전에 알림을 받을 수 있습니다.

- 일정 시작 10분 전
- 일정 시작 5분 전

알림이 오면 말풍선이 강조되어 표시되고, 고양이 울음소리가 한 번 재생됩니다.

## 자주 묻는 질문

### 다운로드 받고 난 후 macOS 보안 경고가 떠요

처음 실행할 때 macOS에서 아래와 같은 안내가 나올 수 있습니다.
> “Time Tracker”을(를) 열지 않음

실행 방법:

1. 안내창에서 `완료`를 누릅니다.
2. Mac의 `시스템 설정`을 엽니다.
3. `개인정보 보호 및 보안`으로 이동합니다.
4. 아래쪽에 `Time Tracker 사용이 차단됨` 문구가 보이면 `그래도 열기`를 누릅니다.
5. 다시 확인 창이 뜨면 `열기`를 누릅니다.

### 다른 사람이 내 캘린더를 볼 수 있나요?

아니요.
Google Calendar 권한은 사용자 본인의 Google 계정으로 직접 승인합니다.
다른 사람이 앱을 사용하더라도 내 캘린더를 볼 수 없습니다.

### 앱이 내 일정을 수정하나요?

아니요.
Time Tracker는 일정을 보여주기 위한 읽기 권한만 사용합니다.

### 앱을 껐다 켜면 캘린더를 다시 연결해야 하나요?

아니요.
한 번 연결하면 연결 상태가 유지됩니다.

### 앱을 완전히 종료하려면 어떻게 하나요?

고양이 위젯 위에서 마우스 오른쪽 버튼을 누른 뒤 `끝내기`를 선택하면 됩니다.

## 참고

Time Tracker는 macOS용 앱입니다.
현재 Windows 버전은 제공하지 않습니다.
