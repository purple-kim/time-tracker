# Time Tracker

작업 시간을 재면서 데스크탑 위에 작은 픽셀 고양이를 띄워두는 앱입니다.
고양이는 노트북을 타이핑하는 모습으로 움직이고, 작업 중에는 타이핑 소리도 들을 수 있습니다.

고양이 테마는 치즈, 까망, 삼색이, 샴 중에서 선택할 수 있습니다.

Google Calendar를 연결하면 다음 일정을 확인하고, 일정 시작 10분 전과 5분 전에 알림을 받을 수 있습니다.

## 앱 다운로드하기

Time Tracker는 macOS 앱 파일로 사용할 수 있습니다.

1. [Time Tracker Releases](https://github.com/purple-kim/time-tracker/releases) 페이지로 이동합니다.
2. 가장 위에 있는 최신 버전을 엽니다.
3. `Time-Tracker-...dmg` 파일을 다운로드합니다.
4. 다운로드한 `.dmg` 파일을 더블 클릭합니다.
5. 열린 창에서 `Time Tracker` 앱을 `Applications` 폴더로 드래그합니다.
6. `Applications` 폴더에서 `Time Tracker`를 실행합니다.

처음 실행할 때 macOS 보안 안내가 나올 수 있습니다.
그 경우 `시스템 설정 > 개인정보 보호 및 보안`에서 앱 실행을 허용해 주세요.

아직 Releases에 파일이 없다면 배포 파일이 만들어지기 전입니다.
개발 중인 버전을 실행하려면 아래의 `개발용으로 실행하기`를 참고해 주세요.

## 앱 조작하기

고양이 위젯 위에서 마우스 오른쪽 버튼을 누르면 메뉴가 열립니다.

메뉴에서 할 수 있는 일은 아래와 같습니다.

- `고양이 테마`: 치즈, 까망, 삼색이, 샴 중 하나를 선택합니다.
- `타이핑 소리`: 타이핑 소리를 켜거나 끕니다.
- `캘린더 연결`: Google Calendar를 연결합니다.
- `캘린더 연결 해제`: 이미 연결된 Calendar를 해제합니다.
- `시작`: 작업 시간을 재기 시작합니다.
- `일시정지`: 작업 시간을 잠시 멈춥니다.
- `끝내기`: 앱 실행을 종료합니다.

고양이를 드래그하면 원하는 위치로 옮길 수 있습니다.

## Google Calendar 연결하기

Google Calendar 연결은 선택 기능입니다.
캘린더를 연결하지 않아도 타이머와 고양이 위젯은 사용할 수 있습니다.

별도 설정 파일을 만들 필요 없이 앱에서 바로 연결할 수 있습니다.

1. 앱을 실행합니다.
2. 고양이 위젯 위에서 마우스 오른쪽 버튼을 누릅니다.
3. `캘린더 연결`을 선택합니다.
4. Google 로그인 화면이 열리면 본인 계정을 선택합니다.
5. 경고 화면이 나오면 `계속`을 누릅니다.
6. Calendar 읽기 권한을 허용합니다.
7. 연결이 완료되면 다음 일정이 말풍선에 표시됩니다.

앱은 일정을 수정하지 않습니다.
다음 일정을 보여주기 위해 읽기 권한만 사용합니다.

한 번 연결하면 앱을 껐다가 다시 켜도 연결 상태가 유지됩니다.

### 개발자가 다른 Google OAuth 설정을 쓰고 싶다면

기본 앱 설정 대신 직접 만든 Google OAuth Client ID를 쓰고 싶을 때만 `google-calendar-config.json` 파일을 사용할 수 있습니다.

프로젝트 폴더 안에 있는 `google-calendar-config.example.json` 파일을 복사한 뒤, 파일 이름을 아래처럼 바꿉니다.

```text
google-calendar-config.json
```

그리고 본인이 발급받은 값을 넣습니다.

```json
{
  "clientId": "여기에_클라이언트_ID",
  "clientSecret": ""
}
```

이 파일은 개인 설정 파일입니다. GitHub에 올리지 마세요.

## 개발용으로 실행하기

아래 내용은 앱을 수정하거나 직접 빌드하려는 사용자를 위한 안내입니다.
일반 사용자는 위의 `앱 다운로드하기`만 따라 해도 됩니다.

### 1. 프로젝트 내려받기

1. [Time Tracker GitHub 페이지](https://github.com/purple-kim/time-tracker)에 접속합니다.
2. 초록색 `Code` 버튼을 누릅니다.
3. `Download ZIP`을 선택합니다.
4. ZIP 파일 압축을 풉니다.
5. 압축이 풀린 `time-tracker-main` 폴더를 찾기 쉬운 곳에 둡니다.

### 2. 터미널 열기

1. 키보드에서 `Command` + `Space`를 누릅니다.
2. 검색창에 `터미널` 또는 `Terminal`을 입력합니다.
3. `Terminal` 앱을 실행합니다.

터미널이 열리면 다운로드한 폴더로 이동합니다.

가장 쉬운 방법은 아래 순서입니다.

1. 터미널에 `cd `를 입력합니다. `cd` 뒤에 공백이 있어야 합니다.
2. 압축을 푼 `time-tracker-main` 폴더를 터미널 창 안으로 끌어다 놓습니다.
3. `Enter`를 누릅니다.

예를 들어 `Downloads` 폴더에 압축을 풀었다면 아래처럼 보일 수 있습니다.

```bash
cd ~/Downloads/time-tracker-main
```

### 3. 필요한 파일 설치하기

처음 한 번만 아래 명령어를 실행합니다.

```bash
npm install
```

설치가 끝날 때까지 잠시 기다립니다.

### 4. 개발용으로 실행하기

```bash
npm start
```

이 방식은 개발용 실행입니다.
터미널을 닫으면 앱도 함께 종료될 수 있습니다.

### 5. macOS 앱 파일 만들기

내 컴퓨터에서 직접 `.dmg` 앱 파일을 만들고 싶다면 아래 명령어를 실행합니다.

```bash
npm run dist:mac
```

완료되면 `dist` 폴더 안에 `.dmg` 파일이 생성됩니다.

## 배포하기

이 저장소는 버전 태그를 올리면 GitHub Actions가 자동으로 macOS `.dmg` 파일을 만들고 Release에 첨부합니다.

예시:

```bash
git tag v0.1.0
git push origin v0.1.0
```

태그가 올라가면 GitHub의 `Actions` 탭에서 빌드가 실행됩니다.
빌드가 끝나면 `Releases` 페이지에서 `.dmg` 파일을 다운로드할 수 있습니다.

## 참고

- Google Calendar 권한 토큰은 각 사용자 컴퓨터에만 저장됩니다.
- 앱이 제대로 실행되지 않으면 개발용 프로젝트 폴더에서 `npm install`을 다시 실행한 뒤 `npm start`를 실행해 보세요.
