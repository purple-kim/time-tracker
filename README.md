# Time Tracker

작업 시간을 재면서 데스크탑 위에 작은 픽셀 고양이를 띄워두는 앱입니다.
고양이는 노트북을 타이핑하는 모습으로 움직이고, 작업 중에는 타이핑 소리도 함께 들을 수 있습니다.

고양이 테마는 치즈, 까망, 삼색이, 샴 중에서 선택할 수 있습니다.

Google Calendar를 연결하면 다음 일정을 확인하고, 일정 시작 10분 전과 5분 전에 알림을 받을 수 있습니다.

## 준비물

이 앱은 GitHub에서 파일을 내려받아 직접 실행하는 방식입니다.

먼저 아래 항목이 필요합니다.

- macOS
- Node.js: 앱 실행에 필요한 프로그램입니다.
- Google 계정: Google Calendar를 연결할 때만 필요합니다.

Node.js가 없다면 [Node.js 공식 사이트](https://nodejs.org/)에서 `LTS` 버전을 설치해 주세요.

Google Calendar를 연결하지 않아도 타이머와 고양이 위젯은 사용할 수 있습니다.

## 1. 앱 다운로드하기

GitHub에서 앱 실행에 필요한 파일을 다운로드합니다.

1. [Time Tracker GitHub 페이지](https://github.com/purple-kim/time-tracker)에 접속합니다.
2. 화면 오른쪽 위의 초록색 `Code` 버튼을 누릅니다.
3. `Download ZIP`을 선택합니다.
4. 다운로드된 ZIP 파일을 엽니다.
5. 압축이 풀리면 `time-tracker-main` 폴더가 생깁니다.
6. 이 폴더를 `Downloads`나 `Desktop`처럼 찾기 쉬운 곳에 둡니다.

다음 단계에서 이 폴더 위치가 필요합니다.

## 2. 실행 준비하기

처음 한 번만 앱 실행에 필요한 파일을 설치하면 됩니다.

먼저 터미널을 엽니다.

1. 키보드에서 `Command` + `Space`를 누릅니다.
2. 검색창이 열리면 `터미널` 또는 `Terminal`을 입력합니다.
3. `Terminal` 앱을 실행합니다.

터미널이 열리면, 방금 다운로드한 폴더로 이동합니다.

가장 쉬운 방법은 아래 순서입니다.

1. 터미널에 `cd `를 입력합니다. `cd` 뒤에 공백이 있어야 합니다.
2. 압축을 푼 `time-tracker-main` 폴더를 터미널 창 안으로 끌어다 놓습니다.
3. `Enter`를 누릅니다.

예를 들어 `Downloads` 폴더에 압축을 풀었다면 아래처럼 보일 수 있습니다.

```bash
cd ~/Downloads/time-tracker-main
```

폴더 이동이 끝났다면 아래 명령어를 입력합니다.

```bash
npm install
```

설치가 끝날 때까지 잠시 기다립니다.
완료되면 다음 단계에서 앱을 실행할 수 있습니다.

## 3. 앱 실행하기

터미널에서 아래 명령어를 입력합니다.

```bash
npm start
```

데스크탑 위에 고양이 위젯이 나타나면 실행된 것입니다.

앱을 다시 실행하고 싶을 때도 같은 폴더에서 아래 명령어를 입력하면 됩니다.

```bash
npm start
```

터미널을 닫으면 앱도 함께 종료될 수 있습니다.
앱을 사용하는 동안에는 터미널 창을 열어두는 것을 권장합니다.

## 4. 앱 조작하기

고양이 위젯 위에서 마우스 오른쪽 버튼을 누르면 메뉴가 열립니다.

메뉴에서 할 수 있는 일은 아래와 같습니다.

- `고양이 테마`: 치즈, 까망, 삼색이, 샴 중 하나를 선택합니다.
- `타이핑 소리`: 타이머가 시작된 상태에서만 ON/OFF를 바꿀 수 있습니다.
- `캘린더 연결`: Google Calendar를 연결합니다.
- `캘린더 연결 해제`: 이미 연결된 Calendar를 해제합니다.
- `시작`: 작업 시간을 재기 시작합니다.
- `일시정지`: 작업 시간을 잠시 멈춥니다.
- `끝내기`: 앱 실행을 종료합니다.

고양이를 드래그하면 원하는 위치로 옮길 수 있습니다.

## 5. Google Calendar 설정하기

Google Calendar를 연결하려면 본인 Google 계정으로 OAuth 설정을 만들어야 합니다.

캘린더를 연결하지 않고 타이머만 사용할 경우 이 단계는 건너뛰어도 됩니다.

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속합니다.
2. 새 프로젝트를 만들거나 기존 프로젝트를 선택합니다.
3. `API 및 서비스`에서 `Google Calendar API`를 사용 설정합니다.
4. `OAuth 동의 화면`을 설정합니다.
5. 사용자 유형은 `외부`를 선택합니다.
6. 테스트 사용자에 본인 Google 이메일을 추가합니다.
7. `사용자 인증 정보`에서 `OAuth 클라이언트 ID`를 만듭니다.
8. 애플리케이션 유형은 `데스크톱 앱`을 선택합니다.
9. 생성된 `클라이언트 ID`와 `클라이언트 보안 비밀번호`를 복사합니다.

## 6. 캘린더 설정 파일 만들기

다운로드한 프로젝트 폴더 안에 있는 `google-calendar-config.example.json` 파일을 복사합니다.

복사한 파일 이름을 아래처럼 바꿉니다.

```text
google-calendar-config.json
```

`google-calendar-config.json` 파일을 열고 아래 값들을 본인이 발급받은 값으로 바꿉니다.

```json
{
  "clientId": "여기에_클라이언트_ID",
  "clientSecret": "여기에_클라이언트_보안_비밀번호"
}
```

이 파일은 개인 설정 파일입니다. GitHub에 올리지 마세요.

## 7. Google Calendar 연결하기

1. 앱을 실행합니다.
2. 고양이 위젯 위에서 마우스 오른쪽 버튼을 누릅니다.
3. `캘린더 연결`을 선택합니다.
4. Google 로그인 화면이 열리면 본인 계정을 선택합니다.
5. 경고 화면이 나오면 `계속`을 누릅니다.
6. Calendar 읽기 권한을 허용합니다.
7. 연결이 완료되면 다음 일정이 말풍선에 표시됩니다.

앱은 일정을 수정하지 않습니다. 다음 일정을 보여주기 위해 읽기 권한만 사용합니다.

한 번 연결하면 앱을 껐다가 다시 켜도 연결 상태가 유지됩니다.

## 참고

- Google Calendar 권한 토큰은 각 사용자 컴퓨터에만 저장됩니다.
- 앱이 제대로 실행되지 않으면 `npm install`을 다시 실행한 뒤 `npm start`를 실행해 보세요.
